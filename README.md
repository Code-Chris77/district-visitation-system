# District Visitation Management System (DVMS)

A web application designed for The Church of Pentecost to improve pastoral visitation by securely managing member information, GPS coordinates, visitation schedules, and navigation.

## Features

- Role-based authentication
- Member management
- GPS coordinate capture
- Interactive maps
- Visitation scheduling
- Reports and analytics

## Technology Stack

- Next.js
- NestJS
- PostgreSQL
- Prisma
- TypeScript
- Tailwind CSS
- OpenStreetMap
